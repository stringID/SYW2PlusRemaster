# HQ
HQ 파일은 '&#42;.HQ 및 &#42;' 파일의 두 쌍으로 구성되어 있습니다.
> 예) Data.hq 및 Data 파일

여기서 &#42;.HQ 파일은 헤더 파일, &#42; 파일은 데이터 파일로 분류됩니다.
이렇게 두 쌍으로 되어있는 이 파일들은 기본적으로 암호화가 되어 있어서 먼저 복호화를 해주어야 합니다.
복호화 공식은 다음과 같습니다 : 
<pre>
	<code>
		(byte & 16) ? byte - 16 : byte + 16
	</code>
</pre>

복호화를 한 뒤, 헤더 파일을 참조하여 데이터 파일의 압축을 해제하면 됩니다.
헤더 파일의 구조는 다음과 같습니다 : 
<pre>
	<code>
		총라인수 : 
		repeat(&#91;파일개수&#93;->)
		{
			파일명\t
			&#91;파일크기&#93;\t
			&#91;파일오프셋&#93;
			->
		}
	</code>
</pre>
